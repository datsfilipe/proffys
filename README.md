## Prerequisites

- Node >= 12.18.3

## How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/datsfilipe/proffys.git
```

2. Install the dependencies.

```bash
npm install
```

3. Run the project.

```bash
npm run dev
```
